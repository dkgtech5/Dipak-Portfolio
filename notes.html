Cybersecurity Notes

Below are some key notes and summaries from my learning journey in cybersecurity and ethical hacking.

Linux Command Basics

  • ls – List directory contents
  • cd – Change directory
  • chmod – Change file permissions
  • netstat – Display network connections

Nmap Scan Types

  • nmap -sS – TCP SYN scan
  • nmap -sV – Version detection
  • nmap -A – Aggressive scan (OS, version, script)
  • nmap -T4 – Faster execution

OWASP Top 10

  • 1. Injection
  • 2. Broken Authentication
  • 3. Sensitive Data Exposure
  • 4. Security Misconfiguration
  • 5. XSS – Cross-site Scripting

Tools & Usage

  • Burp Suite – Intercept and modify web traffic
  • Metasploit – Exploit development and testing
  • Wireshark – Packet capture and analysis
  • Hydra – Brute force login credentials
